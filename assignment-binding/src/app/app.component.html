Bind it!

  1. Create the new components GameControl, Odd and Even
  2. The GameControl Component should have buttons to start and stop the game
  3. When starting the game, an event (holding an incrementing number) should get emitted each second (ref = setInterval())
  4. A new Odd component should get created for every odd number emitted, the same should happen for the Even Component (on even numbers).
  5. Simply output Odd - Number, or Even - Number in the two components
  6. Style the element (eg, paragraph) holding your output text differently in both components.
